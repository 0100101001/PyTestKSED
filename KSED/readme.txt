
Описание каталога:

KSED - Корневая директория тест-сьюта;
Pages - Директория содержащая файлы с Page Object;
    PageObject - Основной файл с Page Object;
TestData - Директория содержащая фалы с тестовыми данными и локаторами;
    data - Файл с тестовыми данными (ШК документов, урлы и т.д.) файл необходимо актуализировать перед прогоном тестов;
Tests - Директория содержащая тест-кейсы;
Utils - Директория содержащая файлы вспомогательных функций и фикстур;
WebDriver - Директория содержащая драйверы браузеров;

conftest - Конфигурационный файл;

test_Runner - Файл запуска тестов (тест-сьюта)
