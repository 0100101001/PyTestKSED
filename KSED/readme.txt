
Описание каталога:

KSED - Корневая директория тест-сьюта;
Pages - Директория содержащая файлы с Page Object;
    PageObject.py - Основной файл с Page Object;
TestData - Директория содержащая фалы с тестовыми данными и локаторами;
    locators.py - Локаторы;
    data.py - Файл с тестовыми данными (ШК документов, урлы и т.д.) файл необходимо актуализировать перед прогоном тестов;
Tests - Директория содержащая тест-кейсы;
Utils - Директория содержащая файлы вспомогательных функций и фикстур;
WebDriver - Директория содержащая драйверы браузеров;

conftest.py - Конфигурационный файл;

test_Runner.py - Файл запуска тестов (тест-сьюта)

requirements.txt - Перечень необходимых пакетов
